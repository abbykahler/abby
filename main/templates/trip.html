Trip Tracker

Trip Tracker is a simple app that can keep track of your next trip. The landing page has a registration and login with valdiations to make sure the user can access their account.

The home page will display the users itinerary after the trip is created.

The create page allows the user to customize their trip.

The travel plan is now updated on the home screen and the trip can be removed or edited.

All the details of the trip can be viewed on the about page.
